# Maze Runner Pro 🎮
 
Maze Runner Pro is a Python game built using Pygame.

## Features 

* Random Maze Generation using Kruskal Algorithm
* Shortest Path Hint using Dijkstra Algorithm
* Coin Collection System
* Multiple Levels
* Score System
* Attractive UI

## Technologies Used

* Python
* Pygame
* Heapq
* Random
* Time

## How to Run

```bash
pip install pygame
python pro.py
```

## Controls

* Arrow Keys / WASD → Move Player
* H → Show Hint Path
* R → Restart Level
* N → Next Level
* ESC → Quit
